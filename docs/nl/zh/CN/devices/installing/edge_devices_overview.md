# 边缘设备概述
{: #edge_devices_overview}

边缘设备提供了企业或服务提供者核心网络的入口点。 示例包括智能手机、监控摄像头，甚至已连接因特网的微波炉。

IBM Edge Application Manager (IEAM) 可用于基础结构或服务器，包括分布式设备。

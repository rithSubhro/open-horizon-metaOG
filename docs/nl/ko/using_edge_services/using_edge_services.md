# 에지 서비스 사용
{: #using_edge_services}

{{site.data.keyword.edge_notm}}({{site.data.keyword.ieam}})는 중앙 집중식 서비스인 Exchange를 포함합니다. 이 서비스는 에이전트 및 agbot의 완전 자율 작업을 지원합니다. 또한 에지 서비스를 사용하면 연관된 배치 패턴을 작성하고 해당 배치 패턴을 실행하기 위해 에지 노드를 등록할 수 있습니다.

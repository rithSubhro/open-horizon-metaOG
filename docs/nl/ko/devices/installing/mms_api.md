# MMS REST API
{: #mms_rest_apis}

MMS(Model Management System)에는 두 개의 API(클라우드 동기화 서비스[CSS] 및 에지 동기화 서비스[ESS])가 있습니다. [에지 동기화 서비스](../developing/mms_swagger.json)를 참조하십시오. 이는 에지 동기화 서비스의 기본 패키지입니다.
